基本設定

翻訳 API（任意）

内蔵辞書に無い文や文章を翻訳するために、任意の翻訳 API を利用できます。
注意: 有効化すると、ページ上のテキストが選択した提供者に送信されます。機密情報を含む場合はオフにしてください。

使い方のヒント

  • この拡張は GitHub の UI をなるべく安全に日本語化します。コード領域や差分は翻訳しません。
  • キーボード: Alt + J で有効・無効を切り替え。
  • 右上の拡張アイコンから「今すぐ翻訳」を実行できます。
プライバシー: 翻訳 API を有効化すると、ページ中のテキストが外部の翻訳サービスに送信されます。自己責任でご利用ください。
